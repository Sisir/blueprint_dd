h1. Blueprint DropDown Readme

This is a simple extension to "blueprintcss":http://blueprintcss.org

requires: "blueprintcss tabs":http://github.com/Montoya/blueprint-plugin---tabs/tree/master


h2. Setup Instructions

You should/must have blueprint css and blueprint tabs installed

# get this out of github and place the whole blueprint_dd directory into your stylesheets directory.
# add the following to your html (eg your application layout):
<pre>
<link rel="stylesheet" href="/stylesheets/blueprint_dd/screen.css" type="text/css" media="screen" />  
<!--[if IE]>
  <link rel="stylesheet" href="/stylesheets/blueprint_dd/ie.css" type="text/css" media="screen, projection" />
<![endif]-->
</pre>

Good luck!

It's not perfect, so if you make it better let me know!
